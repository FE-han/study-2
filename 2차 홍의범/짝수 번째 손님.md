## 문제 설명

- 짝수 번째 손님들을 뽑아내기<br>


## 자료 구조
- n
타입 : list<br>
저장 데이터 : my_list [5,4,2,1] <br>

- n_list 
타입 : list 
저장 데이터 : 짝수손님을 추가시킬 배열 <br>

## 풀이 과정
예시 :
```txt
1. for문 으로 사람 수 만큼 반복을 돌리고
2. 2로 나눈 나머지가 짝수일때 append로 추가시킴 
```

## 코드 구현
```python
# 풀이과정 1

if 1 <= len(n) <= 100:
    for i in range(1, len(n)+1):
        if i % 2 == 0:
            n_list.append(n[i-1])
            
print(n_list)

```
