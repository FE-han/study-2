
## 평균 구하기!

### 문제 설명

- 0이 나올때까지 숫자를 입력받고, 평균값을 구하는 앱<br>


### 자료 구조

- my_list<br>
    - 타입 : 리스트
    - 저장 데이터 : 입력받을 숫자들 저장

- num<br>
    - 타입 : 숫자
    - 저장 데이터 : 숫자 입력
### 풀이 과정

>-문제를 풀기 위한 풀이 과정을 작성합니다.<br>
>-풀이 과정을 단계별로 나누어 적거나, 의사 코드(pseudo code) 형태로 작성합니다.<Br>

```txt
1. 변수 my_list 생성

2. 반복문 시작하고, 변수 num에 숫자로 입력 받음

3. 변수 num이 0이 들어오면 반복문 빠져나옴

4. num이 0 아니면 리스트 변수 my_list에 계속 저장

5. 전체 합을 구하고 my_list의 길이로 나눠서 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
my_list = []

# 풀이과정 2
while True:
    num = int(input())

# 풀이과정 3
if(num == 0):
    break

# 풀이과정 4
else:
    my_list.append(num)

# 풀이과정 5
print(sum(my_list) / len(my_list))

```


점수 : 100점 / 100점
