
## 계단 별자리가 점점 커지고있어요!

### 문제 설명

- 입력 받은 숫자까지 별 하나씩 증가하며 출력<br>


### 자료 구조

- num<br>
    - 타입 : 숫자
    - 저장 데이터 : 묵표 별자리 숫자
### 풀이 과정

>-문제를 풀기 위한 풀이 과정을 작성합니다.<br>
>-풀이 과정을 단계별로 나누어 적거나, 의사 코드(pseudo code) 형태로 작성합니다.<Br>

```txt
1. 변수 num에 숫자 입력

2. 1부터 num+1까지 반복, 하나씩 증가하며 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
num = int(input())

# 풀이과정 2
for i in range(1, num+1):
    print('*' * i)

```
