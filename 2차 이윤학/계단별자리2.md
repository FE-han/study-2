## 계단 별자리 II (계단 별자리가 점점 커지고있어요!)

### 문제 설명

- 이번에는 입력한 수에 따라 계단 별자리를 출력하는 프로그램<br>



### 자료 구조

- num<br>
    - 타입 : int
    - 저장 데이터 : int(input())



### 풀이 과정
예시 :
```txt
1. 숫자입력 받기

2. 입력 받은 만큼 "*" 출력



```

### 코드 구현
```python
# 풀이과정 1
num = int(input())

# 풀이과정 2
for a in range(num+1):
    print("*"*a)
```
